# Django Blog Project

## Introducció

Aquest projecte consisteix en la creació d’un blog amb Django.

L’objectiu és aprendre:
- Django
- Routing
- Templates
- Models
- Base de dades
- GitHub Actions

---

## Instal·lació ràpida

### Clonar repositori

```bash
git clone URL_REPOSITORI
```

---

### Entrar projecte

```bash
cd Porjecte
```

---

### Crear entorn virtual

```bash
python -m venv venv
```

---

### Activar entorn virtual

```bash
venv\Scripts\activate
```

---

### Instal·lar Django

```bash
pip install django
```

---

### Migracions

```bash
python manage.py migrate
```

---

## Execució

```bash
python manage.py runserver
```

---

```text
http://127.0.0.1:8000
```

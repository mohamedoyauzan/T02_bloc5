## Instal·lació ràpida

### Clonar repositori

```bash
git clone https://github.com/mohamedoyauzan/T02_bloc5.git
```

---

### Entrar projecte

```bash
cd Porjecte
```

---

### Crear entorn virtual

```bash
python -m venv venv
```

---

### Activar entorn virtual

```bash
venv\Scripts\activate
```

---

### Instal·lar Django

```bash
pip install django
```

---

### Migracions

```bash
python manage.py migrate
```

---

## Execució

```bash
python manage.py runserver
```
